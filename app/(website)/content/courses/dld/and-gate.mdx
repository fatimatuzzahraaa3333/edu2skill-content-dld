The AND gate produces outputs HIGH only when all inputs are HIGH. It performs logical multiplication in digital electronics.


## 1. Properties
<ol type="a" style={{ listStyleType: "lower-alpha", paddingLeft: "20px" }}>
    <li>Minimum two inputs</li>
    <li>Output is 1 only if every input is 1</li>
    <li>Output is 0, if any input is 0</li>
</ol>
