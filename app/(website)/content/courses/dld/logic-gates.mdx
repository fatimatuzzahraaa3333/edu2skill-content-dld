Logic gates are basic building blocks of <mark>combinational-logic-circuit</mark>. A combinational-logic-circuit is a digital circuit whose output depends only on the present inputs. It does not store any past information and does not use memory elements.

## 1. Types of Logic Gates
There are following <mark>seven</mark> logic gates.
<ol type="a" style={{ listStyleType: "lower-alpha", paddingLeft: "20px" }}>
    <li>AND Gate</li>
    <li>OR Gate</li>
    <li>NOT Gate</li>
    <li>NAND Gate</li>
    <li>NOR Gate</li>
    <li>XOR Gate</li>
    <li>XNOR Gate</li>
</ol>

## 2. Basic Logic Gates
There are following basic logic gates (first three). These are so called <mark>basic-logic-gates</mark> because we can construct rest of the gates using these basic gates.

    <li>AND Gate</li>
    <li>OR Gate</li>
    <li>NOT Gate</li>
