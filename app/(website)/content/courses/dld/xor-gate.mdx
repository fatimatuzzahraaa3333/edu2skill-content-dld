This gate will generate otput as 1 (HIGH), if number of 1's in inputs are ODD. This is also called <mark>ODD Function</mark>.


## 1. Properties

<ol type="a" style={{ listStyleType: "lower-alpha", paddingLeft: "20px" }}>
    <li>Output will be 1, if number of 1's in inputs are ODD. Otherwise, output is 0.</li>
    <li>This is also called <mark>ODD Function</mark></li>
</ol>

<div className="text-center"> Y = AB′ + A′B = A ⊕ B </div>
