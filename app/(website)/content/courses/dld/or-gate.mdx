The OR gate produces a HIGH output when any input is HIGH. It performs logical addition in combinational logic circuits.


## 1. Properties
<ol type="a" style={{ listStyleType: "lower-alpha", paddingLeft: "20px" }}>
    <li>Minimum two inputs</li>
    <li>Output is 1, if any input is 1</li>
    <li>Output is 0, only if all inputs are 0</li>
</ol>
